⚙️ SETUP & DEPLOYMENT
═══════════════════════════════════════════════════════════════════════════════

QUICK SETUP:
  1. Create .env.local with: NEXT_PUBLIC_GEMINI_API_KEY=your_key
  2. Run: pnpm install && pnpm dev
  3. Open: http://localhost:3000

GET API KEY:
  1. Go to: https://aistudio.google.com
  2. Click "Get API Key"
  3. Create or select project
  4. Copy and paste into .env.local

PRODUCTION BUILD:
  pnpm build
  pnpm start

DEPLOY TO VERCEL:
  vercel
